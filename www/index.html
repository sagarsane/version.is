version.is

| An API for getting the latest version of Open Source projects

Usage

A few examples:

GET http://api.version.is/jquery
  -> 1.8.2pre
GET http://api.version.is/jquery?format=json
  -> {"project":"jquery", "version":"1.8.2pre"}
GET http://api.version.is/jquery?format=json&callback=foo
  -> foo({"project":"jquery", "version":"1.8.2pre"});
